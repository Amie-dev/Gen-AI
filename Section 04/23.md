
# **23. Chat Models vs Tools (Quick & Clear)**

> Chat models **think and talk**.
> Tools **do and fetch**.

---

### What they do

* Understand natural language
* Reason about problems
* Generate responses
* Decide **whether a tool is needed**

### Strengths

* Language understanding
* Reasoning & planning

1. User asks a question
2. Chat model decides
3. Tool is called (if needed)
4. Tool returns data
5. Chat model explains result

* Tools alone cannot reason
