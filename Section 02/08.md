# 08: Chains vs Agents

This section explains the **difference between Chains and Agents**, a very important concept in **LangChain, AI agents, exams, and interviews**.

---

## 1. What is a Chain?

A **Chain** is a **fixed, predefined sequence of steps** where:

* Input goes through a set order
* Each step executes exactly as designed
* No decision-making happens during execution

## 2. What is an Agent?

An **Agent** is a **dynamic decision-making system** that:

* Thinks before acting
* Chooses tools dynamically
* Decides next steps based on results

ğŸ“Œ **Simple definition:**

> An Agent is an intelligent system that can decide *what to do next* to achieve a goal.

## 7. When to Use a Chain

Use **Chains** when:

* Steps are simple and fixed
* Task is predictable
* No decision-making is required

Examples:

* Text summarization
* Translation
* Formatting data

---

## 8. When to Use an Agent

Use **Agents** when:

* Task is complex
* Requires multiple tools
* Steps cannot be predefined

Examples:

* Research assistant
* Customer support bot
* Autonomous AI systems
